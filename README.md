# journal-pc-home

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件形式存储，可直接通过浏览器访问，不依赖后端服务。

## 项目简介

这是一个以 HTML 页面归档为主要目的的仓库。所有页面均为独立文件，无外部链接依赖，便于离线保存与查阅。仓库内容不限定于特定主题或网站，仅作为结构化存储和展示的载体。

## 目录结构

```
journal-pc-home/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 归档页面存放目录
│   ├── example1.html
│   └── example2.html
└── assets/             # 公共资源（CSS / 图片等）
    └── style.css
```

- `pages/`：存放主要归档的 HTML 页面，每页独立。
- `assets/`：存放页面引用的样式、图片等资源文件。
- `index.html`：仓库根目录下的入口文件，用于导航或展示说明。

## 页面归档说明

- 所有页面均为纯前端静态 HTML，不包含动态数据请求。
- 页面内容可能随时间更新或补充，历史版本可通过 Git 提交记录追溯。
- 页面命名建议采用有意义的英文或拼音，避免特殊字符。

## 维护说明

- 新增页面：在 `pages/` 下创建 `.html` 文件，并确保引用资源路径正确。
- 更新页面：直接修改对应文件，提交时请附带简要说明。
- 删除页面：请确认无其他页面引用后，移除文件并提交。
- 不建议在页面中嵌入外部脚本或敏感信息，以保持仓库的通用性和安全性。

## 贡献

欢迎通过 Pull Request 提交页面归档或改进。请确保提交的内容不违反法律法规，并保持文件结构清晰。
